# TasqueServices
Серверная часть сервиса отслеживания выполнения задач. Проект сделан в учебных целях.
В проекте используются: 
* REST api для создания, редактирования и получения задач с пагиацией; 
* Мультисервисная архитектура с пересылкой сообщений с помощью RabbitMQ; 
* Отлавливание исключений и запись метрик с помощью Prometheus и OpenTelemetry;
* Хранение данных с датабазой Postgres.

## Инструкция по запуску сервиса в контейнере
1. Чтобы запустить сервис из контейнера необходим Docker и Docker Dersktop
2. Зайти через консоль в корневую папку проекта и ввести команду:
```bash
docker-compose up -d
```

Для отладки из Visual Studio Solution достаточно запустить контейнеры "prometheus", "postgres_db" и "rabbitmq".
